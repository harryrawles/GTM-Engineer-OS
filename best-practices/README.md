# best-practices/

Best-practice references and checklists. Part of the **shared layer**, available to every client.

## What's here today

`best-practices/playbooks/` - 26 reusable, client-agnostic campaign playbooks (uploaded ColdIQ material,
e.g. funding, hiring signal). Start-points for any client's campaign, adapt per client, never use verbatim
(`CLAUDE.md` → *Source Knowledge*).

`best-practices/growth-plays.md` - client-agnostic tactical plays (re-run non-repliers, add a sequence
step) for squeezing more value out of an existing list without fresh data. Used by
`gtm-skills/client-checkin-writer.md`, can also be pulled manually.

## What lands in this folder going forward

Outside `playbooks/`, this directory is the **promotion destination** for new, de-identified best-practice
patterns that emerge from client work - written here by `gtm-skills/weekly-reviewer.md` (Step 8, on
`Confirmed` patterns) and `gtm-skills/client-offboarder.md` (STEP 3). Drop new files here and reference
them by path; Claude auto-discovers them.

**Rule:** nothing in this folder may contain a specific client's private data. Best practices are universal.

See root `CLAUDE.md` → *Shared Layer* and *The Compounding Loop* for how this fits the OS.
