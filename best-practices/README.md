# best-practices/

Best-practice references and checklists. Part of the **shared layer**, available to every client.

Drop new best-practice files here and reference them by path from any skill or session; Claude
auto-discovers them. Content is uploaded later.

**Rule:** nothing in this folder may contain a specific client's private data. Best practices are universal.

See root `CLAUDE.md` → *Shared Layer* for how this fits the OS.
