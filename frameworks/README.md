# frameworks/

Strategic frameworks and mental models — part of the **shared layer**, available to every client on the
same footing as skills.

Drop new framework files here and reference them by path from any skill or session; Claude auto-discovers
them. Content is uploaded later.

**Rule:** nothing in this folder may contain a specific client's private data. Frameworks are universal.

See root `CLAUDE.md` → *Shared Layer* for how this fits the OS.
